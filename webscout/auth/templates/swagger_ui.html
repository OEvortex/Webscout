Chat Completions

POST /v1/chat/completions
▼

Create a chat completion with various AI models. Supports both streaming and non-streaming responses.

Request Body Parameters

  • model string required
    ID of the model to use
  • messages array required
    Array of message objects
  • stream boolean
    Whether to stream back partial progress
  • temperature number
    Sampling temperature between 0 and 2
  • max_tokens integer
    Maximum number of tokens to generate

Image Generation

POST /v1/images/generations
▼

Generate images from text prompts using various text-to-image models.

Request Body Parameters

  • prompt string required
    Text description of the desired image
  • model string required
    ID of the image generation model to use
  • n integer
    Number of images to generate (default: 1)
  • size string
    Size of the generated images (e.g., "1024x1024")

Authentication Guide

🔐 API Key Authentication

This API uses API key authentication. Include your API key in the Authorization header of your requests.

Header Format
Authorization: Bearer YOUR_API_KEY

📝 Getting Started

  1. Generate an API key using the /v1/auth/generate-key endpoint
  2. Include the API key in the Authorization header of your requests
  3. Make requests to the protected endpoints

📊 Rate Limits

API keys have configurable rate limits. The default limit is 10 requests per minute.

Example cURL Request
curl -X POST "{{ base_url }}/v1/chat/completions" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-3.5-turbo",
    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'

Code Examples

🐍 Python Example

import requests

url = "{{ base_url }}/v1/chat/completions"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "model": "gpt-3.5-turbo",
    "messages": [
        {"role": "user", "content": "Hello, how are you?"}
    ],
    "stream": False
}

response = requests.post(url, headers=headers, json=data)
print(response.json())

🟨 JavaScript Example

const response = await fetch('{{ base_url }}/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-3.5-turbo',
    messages: [
      {role: 'user', content: 'Hello, how are you?'}
    ],
    stream: false
  })
});

const data = await response.json();
console.log(data);

📱 Node.js with OpenAI SDK

import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: '{{ base_url }}/v1'
});

const completion = await client.chat.completions.create({
  model: 'gpt-3.5-turbo',
  messages: [
    {role: 'user', content: 'Hello, how are you?'}
  ],
});

console.log(completion.choices[0].message.content);
